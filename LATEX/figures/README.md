# Figures — Cross-Cultural Adaptation of WOCAT-SLM

This folder contains figures for the Elsevier article on cross-cultural adaptation.

## Required Figures

1. **adaptation_flowchart.pdf/.eps** — Six-stage flowchart (Beaton et al., 2000):
   - Stage 1: Forward Translation (T1 + T2)
   - Stage 2: Synthesis (T-12)
   - Stage 3: Back-Translation (BT1 + BT2)
   - Stage 4: Expert Committee (8-12 members)
   - Stage 5: Pre-Test (30-40 quilombola farmers + cognitive debriefing)
   - Stage 6: Consolidation (WOCAT-SLM-QBR final version)

2. **study_area_map.pdf/.eps** — Map of Semiárido Nordeste II Territory:
   - Bahia state highlighting 18 municipalities
   - Jeremoabo quilombola communities (11 communities)

3. **equivalence_dimensions.pdf/.eps** — Four equivalence dimensions diagram:
   - Semantic, Idiomatic, Experiential, Conceptual
   - With examples from WOCAT → quilombola context

## Format Notes

- Elsevier prefers **PDF** or **EPS** for vector graphics
- Minimum resolution for raster: 300 DPI
- Maximum width for 3p (three-column) format: ~190mm
